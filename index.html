What's Tenyum-Storage?

Tenyum is a project which wants to help people get benefits form the cloud-computing in China.

Tenyum-Storage is an open-source Web Based Distribution File System we call it TWFS for short.

How Tenyum-Storage (TWFS) works?

TWFS is different with tradtional DFS like Moonfs,HDFS.It's file storage node is provider by dropbox,skydriver and much more online storages.It integrates these storages on the web with tidy and clear APIs.
